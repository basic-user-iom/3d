# Streets GL Alternative Installation

This is an alternative installation of Streets GL, configured to run on port 8081.

## Quick Start

1. Install dependencies (already done):
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```
   Or use the batch file:
   ```bash
   start-dev.bat
   ```

3. Open your browser to: http://localhost:8081

## Configuration

- **Port**: 8081 (to avoid conflicts with other installations)
- **Name**: streets-gl-alt
- **Version**: 2.0.0-alt

## Development

- `npm run dev` - Start development server on port 8081
- `npm run build` - Build for production
- `npm run start` - Serve production build on port 8081

## Notes

This installation is configured to work independently from other Streets GL installations. The port and package name have been changed to avoid conflicts.
