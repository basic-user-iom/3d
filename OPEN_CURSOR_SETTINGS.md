# How to Open Cursor Settings

## Quick Method (Keyboard Shortcut)

**Press `Ctrl+,`** (Control + Comma)

This is the fastest way to open Cursor Settings.

## Alternative Methods

### Method 1: Command Palette
1. Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on Mac)
2. Type: `Preferences: Open Settings`
3. Press Enter

### Method 2: Menu
1. Click **File** → **Preferences** → **Settings**
   - OR: **Edit** → **Preferences** → **Settings** (on Mac)

### Method 3: Direct Settings File
The settings file is located at:
```
C:\Users\Mirjan\AppData\Roaming\Cursor\User\settings.json
```

You can edit this file directly, but it's better to use the Settings UI.

## What to Look For in Settings

Once Settings is open, search for:
- **`sync`** - To find sync settings
- **`chat`** - To find chat-related settings
- **`cloud`** - To find cloud sync settings
- **`account`** - To find account settings

## Check Cloud Sync

1. Open Settings (`Ctrl+,`)
2. Search for: **`sync`** or **`chat history`**
3. Look for:
   - "Settings Sync"
   - "Chat History Sync"
   - "Cloud Sync"
   - "Enable Chat Sync"

If you see these options, you can check if chat history sync is enabled.
