📋 Copy Logs Tool

Simple tool to copy logs for troubleshooting Revit connection

How to use:
  1. Open your browser's Developer Console (Press F12)
  2. Go to the Console tab
  3. Right-click in the console and select "Save as..." OR manually copy logs
  4. Paste the logs in the text area below
  5. Click "Copy All Logs" to copy everything

💡 Tips:

  • Browser Console: Press F12 → Console tab → Right-click → "Save as..."
  • Server Logs: Select text in the command window → Right-click to copy
  • Filter Logs: In browser console, type "Revit" in the filter box to see only Revit-related logs
  • Copy Selected: You can also manually select and copy specific log lines
