# Documentation Archive

Legacy Markdown files live here after we finish their active cleanup pass.
Keeping them grouped by topic lets us preserve context without overwhelming the
repository root.

## Current Folders

Every legacy Markdown file now lives inside one of the archive folders below.
Folder names mirror the original filename prefixes so we can trace a document’s
provenance. Add a new folder when you retire another prefix—keep names
lowercase and kebab-cased for readability.
